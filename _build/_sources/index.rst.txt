TrameGPS Documentation
======================

Bienvenue dans la documentation de TrameGPS, un projet permettant d'analyser les trames GPS en temps réel et de générer des rapports détaillés.

Introduction
------------

Ce projet permet de charger, analyser et visualiser des trames GPS au format NMEA. Il calcule des informations telles que la position la plus au nord, la plus au sud, la distance totale parcourue et génère une page HTML avec les résultats.

Fonctionnalités
---------------

- **Analyse des trames GPS** : Extraction des informations GPS comme la latitude, la longitude, et l'altitude.
- **Calcul des métriques** : Calcul de la distance totale parcourue et des positions extrêmes (nord, sud, est, ouest).
- **Génération de rapports HTML** : Création d'une page HTML pour afficher les résultats et les positions GPS.

Installation
------------

Assurez-vous d'avoir installé Python et les dépendances nécessaires avant d'exécuter le projet.

1. Clonez le repository :
    ```bash
    git clone <repo-url>
    ```
2. Installez les dépendances :
    ```bash
    pip install geopy
    ```

Exemple d'utilisation
----------------------

Voici un exemple d'utilisation de la fonction `analyser_trame` et de la génération de la page HTML avec les résultats :

```python
from tramegps import analyser_trame, generer_page_web

# Analyse des trames à partir du fichier GPS
positions, calculs = analyser_trame('data.gps')

# Générer la page HTML avec les résultats
generer_page_web(positions, calculs, 'output')
